FAQ

Pertanyaan umum.

Bagian ini membantu user baru memahami alur pembayaran, status pesanan, dan ketentuan layanan.

Berapa lama proses top up?

Target proses ditampilkan 1–3 menit setelah pembayaran berhasil terdeteksi. Waktu dapat berubah mengikuti supplier dan antrean pesanan.

Apakah total bayar sudah final?

Ya. Ringkasan pesanan menghitung harga produk, biaya layanan pembayaran, dan total akhir secara realtime sebelum invoice dibuat.

Bagaimana kalau salah User ID?

User harus mengecek ulang User ID dan server sebelum membayar. Catatan keamanan ditampilkan di checkout dan invoice supaya kesalahan input berkurang.

Bagaimana jika sudah bayar tapi belum masuk?

Status order ditampilkan dengan timeline. Gunakan support resmi jika pembayaran berhasil tetapi pesanan belum masuk.

Apakah bisa refund?

Refund mengikuti status pembayaran, status produk, dan ketentuan supplier yang berlaku.

Apakah kode order bisa dicek lagi?

Bisa. Simpan kode order untuk memantau status invoice dan pesanan kapan saja.
